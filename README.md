# 项目简介
角色：普通用户和管理员
技术栈：小程序、Springboot、vue、MySQL数据库
管理员端：web管理系统
用户端：小程序
web管理员端：登录、系统首页、预约管理、位置管理包括楼层管理、自习室管理、座位管理、留言管理、用户管理、公告管理、系统管理包括管理员列表、角色管理、菜单管理。
微信小程序端：登录、注册、系统首页、自习室预约、我的个人中心：编辑个人信息查看余额、系统公告、留言反馈、查看留言列表、退出登录。
自习室预约小程序功能模块概述
（1）注册：用户可以通过注册自习室预约小程序账号再进入。
（2）登录：用户在小程序端登录，用户登录成功即可进入系统，可以进行预订自习室、取消预订和查看个人信息。
（3）预订自习室：用户点击预订即可选择不同楼层和不同时间段进行预订。
（4）入座：用户预订成功后，进入入座页面，可以取消预订和查看预订详情。
（5）我的：用户进入我的模块，编辑个人信息查看余额、系统公告、留言反馈、查看留言列表、退出登录。
自习室预约web后台管理系统功能模块概述
（1）管理员登录：管理员通过管理员账号、密码在web管理端进行登录。
（2）首页：可以查看到预订情况。
（3）位置管理：楼层信息管理，自习室信息管理，座位信息管理。
（4）留言管理：可以查看用户流程和删除用户留言。
（5）用户管理：对系统用户进行管理，删除用户、充值和禁用用户。
（6）公告管理：新增和编辑系统公告。
（7）系统管理：管理员列表管理、角色管理、菜单管理。
